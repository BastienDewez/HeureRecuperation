Outil réservé à l'administrateur

Générateur de codes personnels

Cette page ne fait rien d'autre que calculer, dans votre navigateur, le hachage SHA-256 d'un code personnel. Elle ne l'envoie nulle part. Ajoutez chaque ligne générée dans data/agents.json, puis commitez le fichier.

Rappel : le dépôt est public — n'importe qui peut lire agents.json. Le hachage évite juste que le code apparaisse en clair ; ce n'est pas une protection contre une personne motivée à le deviner. Choisissez des codes à 6 chiffres plutôt que 4 si possible, et ne réutilisez pas un mot de passe important.
